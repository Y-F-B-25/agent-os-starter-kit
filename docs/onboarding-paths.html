Agent OS Setup Paths

Four setup lanes, one destination: a shared vault, clear agent roles, a handoff loop, and quality checks that prove the system works.

Cowork

Chat-first

Best for users who want a friendly Claude session to guide setup.

  1. Paste START_HERE.
  2. Run diagnostic.
  3. Build vault.
  4. Write agent prompts.
Claude Code

Terminal-first

Best for builders who want repo-native setup and command validation.

  1. Open repo.
  2. Run setup prompt.
  3. Create files.
  4. Run checks.
Codex

Implementation-first

Best for OpenAI local agent work, code edits, and browser checks.

  1. Open Codex.
  2. Select Codex lane.
  3. Patch repo.
  4. Verify locally.
Blended

Team-first

Best when Claude and Codex share one vault with clear ownership.

  1. Pick lead.
  2. Define ownership.
  3. Route with VaultBus.
  4. Run QC.

Shared Finish Line

Every path ends with a vault, BRAIN_INDEX, VaultBus, Chief of Staff prompt, specialist prompt, save-up test, eval record, and security review.
